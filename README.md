# Kecik-C3
2mm x 2mm ESP32-C3 MINI 1 Board
Design with EasyEDA


Add module on single PCB design for optimize space 50mm x 50mm 4 layer board

Kecik-C3 Alpha
- Power
- UART_MicroSD
- LED

Kecik-C3 Beta GSM
- Power And SD Card Module
- UART Module
- Sensor with GSM Module

Kecik-C3 Beta RGB
- Power And SD Card Module
- UART Module
- LED Module
